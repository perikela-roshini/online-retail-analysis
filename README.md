# Online Retail Analysis (SQL + Excel Dashboard)

## Project Overview
This project examines a UK-based online retail dataset to find sales trends, customer behavior, and revenue patterns by country.

The project shows skills in:
- SQL data cleaning
- Exploratory data analysis
- Generating business insights
- Creating an interactive Excel dashboard

## Tools Used
- MySQL
- Microsoft Excel

## Project Steps
1. Cleaned raw transactional data using SQL.
2. Removed null values and fixed data types.
3. Created revenue metrics.
4. Ran analytical queries to identify:
   - Monthly revenue trends
   - Top-performing countries
   - Customer purchasing behavior
5. Built an interactive Excel dashboard for visualization.

## Files in this Repository
- **sql/data_cleaning.sql** → Data preparation queries  
- **sql/analysis_queries.sql** → Business analysis queries  
- **Excel/Online_Retail_dashboard.xlsx** → Interactive dashboard  

## Key Insights
- The United Kingdom generated the highest revenue.
- Seasonal spikes occurred during holiday months.
- A small percentage of customers brought in a large portion of revenue.
